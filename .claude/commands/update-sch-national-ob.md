# SCH.FC OB 日本代表追加コマンド

SCH.FC OBが日本代表に選出されたとき、`app/sch/ob/page.tsx` にTrecenCardを追加し
サマリー統計を更新してデプロイするまでを一気通貫で行う。

引数例:
```
/update-sch-national-ob https://www.jfa.jp/national_team/u15_2025/20250513/member.html 對馬羽琉
/update-sch-national-ob 山田太郎 U-17日本代表 2026 クロアチア遠征
```

---

## 実行手順

### Step 1: 選手情報リサーチ（並列起動）

以下の **2エージェントを同時に** 起動する。

#### エージェント A — JFA 公式情報収集
```
以下の選手の日本代表情報を調査してください。

対象選手: $ARGUMENTS に含まれる選手名
JFA参考URL: $ARGUMENTS に含まれるURL（あれば）

取得したい情報:
1. 代表カテゴリ（U-15 / U-16 / U-17 / U-18 / U-20 / フル代表 など）
2. 選出大会・遠征名（例: クロアチア遠征、Vlatko Marković Tournament）
3. 大会期間・開催年
4. 実績（ゴール・MVP等、判明すれば）
5. JFA公式ページのURL

情報源: jfa.jp, web.gekisaka.jp, juniorsoccer-news.com, スポニチ等
調査のみ行い、コードは書かないこと。
```

#### エージェント B — 選手プロフィール・SCH経歴調査
```
以下の選手の経歴を調査してください。

対象選手: $ARGUMENTS に含まれる選手名

取得したい情報:
1. 生年月日（わかれば）
2. 小学生時代の所属クラブ（特にSCH.FCまたはシーハートFCに在籍していた証拠）
3. 現在の所属クラブ（JY/ユース/プロ）
4. SCH.FCを卒業した年度（推定でも可）
5. その他の代表歴・受賞歴

情報源: 所属クラブ公式サイト, transfermarkt, web.gekisaka.jp, SNS等
調査のみ行い、コードは書かないこと。
```

### Step 2: データ確認・整理

A・B の結果をもとに TrecenCard 用データを整理する:

| フィールド | 内容 | 例 |
|-----------|------|-----|
| `year` | 選出年（または年度範囲） | `"2025"`, `"2024-25"` |
| `name` | 選手名（漢字・フルネーム） | `"對馬 羽琉"` |
| `level` | `"日本代表"` または `"ナショナル"` | U-15以上は `"日本代表"` |
| `event` | 代表カテゴリ + 大会名 | `"U-15日本代表選出 / クロアチア遠征"` |
| `note` | SCH経歴 → 現所属 | `"SCH.FC → 川崎フロンターレU-15生田"` |

**level の判定基準:**
- 日本代表（ユースも含む）として国際大会・遠征に出場 → `"日本代表"`
- 国体・地域選抜など代表未満の国内選抜 → `"ナショナル"`

### Step 3: ob/page.tsx を更新

`app/sch/ob/page.tsx` を読み込み、以下2箇所を更新する。

#### 3-1. 日本代表セクション先頭に TrecenCard を挿入

挿入位置: コメント `{/* ─── 日本代表・国際大会歴 ─── */}` の直後の最初のTrecenCardの **前**

```tsx
<TrecenCard
  year="[year]"
  name="[name]"
  level="[level]"
  event="[event]"
  note="[note]"
/>
```

同一選手・同一大会が既に存在する場合は追加しない（重複チェック必須）。

#### 3-2. サマリーグリッドの 🇯🇵 日本代表OB 数を +1

ヘッダー部分の `🇯🇵 日本代表OB` の数字（例: `7+`）を1増やす。

現在のユニーク代表OB選手数を TrecenCard から数えて正確な値に更新すること
（同一選手が複数エントリを持つ場合は1名としてカウント）。

### Step 4: ビルド確認

```bash
npm run build
```

エラーがある場合はその場で修正する。

### Step 5: コミット & デプロイ

```bash
git add app/sch/ob/page.tsx
git commit -m "feat: OB代表ページに[選手名] U-XX日本代表追加"
git push origin master
```

---

## 注意事項

- **SCH OBである確認が取れない選手は追加しない**（ユーザーが「OBである」と明示した場合は信頼する）
- JFA公式サイト（jfa.jp）はスクレイピングブロックされる場合がある → WebSearch で補完する
- 生年月日が取得できなくても TrecenCard の追加自体は可能（note に現所属のみ記載）
- 同一大会への複数選出（複数選手）がある場合は選手ごとに TrecenCard を追加
- `github` リモートへの push は不要（`origin` への push で Vercel に自動デプロイされる）
